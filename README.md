# PanDev Metrics для Eclipse

PanDev Metrics — плагин для Eclipse, который подключает IDE к платформе PanDev Metrics и собирает телеметрию разработки для управленческих метрик и командной аналитики.

## Возможности
- Автоматический сбор активности в IDE без ручных действий
- Метрики продуктивности разработчиков и команд, выявление блокеров
- Аналитика по проектам: прогресс, качество, скорость поставки
- Интеграции с GitLab, Jira и другими системами

## Установка в Eclipse
1. Откройте Help -> Install New Software...
2. Нажмите Add..., в Location вставьте URL на страницу обновления плагина.
3. Отметьте PanDev Metrics и завершите установку.
4. Перезапустите Eclipse, когда будет предложено.

## Оффлайн-установка
Скачайте содержимое каталога releases/1.3.0.v20251202 и в Install New Software укажите file:/<путь>/1.3.0.v20251202.

## Сведения о релизе
- Текущая версия плагина: 1.3.0 (bundle id: pandev-metrics-eclipse-plugin)
- Фича: io.pandev.metrics.feature 1.3.0, поставщик PANDEV
- Назначение: data-driven аналитика разработки, мониторинг командных показателей и эффективности лидов

## Структура репозитория
- Корневой composite указывает на updates/
- updates/1.x/ содержит ветку 1.x; updates/1.x/1.3.x/ собирает релизы 1.3.x
- releases/<version>/ хранит экспортированную p2-площадку для каждого релиза
